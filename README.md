 Wordly Dictionary SPA

Project Overview

Wordly Dictionary is a **Single Page Application (SPA)** built using HTML, CSS, and JavaScript.
The app allows users to search for English words and view their definitions, pronunciation, part of speech, and synonyms—without reloading the page.

This project demonstrates core web development concepts such as API integration, DOM manipulation, and event handling.

---

## Features

*  Search for any English word
*  View:

  * Word
  * Pronunciation (text)
  * Definitions
  * Part of speech
  * Synonyms
*  Error handling for:

  * Invalid words
  * Empty input
*  Dynamic updates (no page refresh)
